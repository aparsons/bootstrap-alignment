Bootstrap Alignment Examples

Responsive Alignment

The following classes allow you to align elements using the bootstrap viewports (xs, sm, md, lg):

Group of classes CSS
.pull-left-* float: left;
.pull-right-* float: right;

Demonstration (Resize your browser)

.pull-right-xs
.pull-right-sm
.pull-right-md
.pull-right-lg

Code

<span class="pull-right-xs">.pull-right-xs</span>

Responsive Text

The following classes allow you to realign text using the bootstrap viewports (xs, sm, md, lg):

Group of classes CSS
.text-left-* text-align: left;
.text-center-* text-align: center;
.text-right-* text-align: right;

Demonstration (Resize your browser)

.text-center-xs
.text-center-sm
.text-center-md
.text-center-lg

Code

<div class="text-center-md">.text-center-md</div>
